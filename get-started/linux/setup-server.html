Setup A Linux Server

This is a simple guide for newbies to set up a Linux(Ubuntu) server.

Update server

After installing a Linux system, updating the server is necessary. Run the following scripts to update server.

  • sudo apt update
  • sudo apt upgrade

Install basic programs.

Some basic programs are useful, but it is up to you whether to install it or not.

  1. Install git
    • $ sudo apt install git
  2. Install pip
    • $ sudo apt install python-pip python-dev build-essential
    • $ sudo pip install --upgrade pip
    • $ sudo export LC_ALL=C
    • $ sudo pip install --upgrade pip
    • $ sudo pip install --upgrade virtualenv
  3. Shadowsocks: to access Google.
    • $ sudo pip install shadowsocks
  4. screen: to run multi processes in the same time.
    • $ sudo apt-get install screen
    • $ screen -S session-name
  5. nodejs: to run NodeJs scripts.
    1. Install from apt
      1. nodejs
        • $ sudo apt-get install nodejs
        • $ sudo ln -s `which nodejs` /usr/bin/node
      2. npm
        • $ sudo apt-get install npm
    2. Download and extract
      • $ wget https://nodejs.org/dist/v6.9.1/node-v6.9.1-linux-x64.tar.xz
      • $ ln -s /opt/nodejs/bin/node /usr/bin/node
      • $ ln -s /opt/nodejs/bin/npm /usr/bin/npm
  6. mongodb
    1. For project development.
      • $ wget https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-ubuntu1604-3.2.9.tgz
      • $ mongod --config <config-file.cfg>
    2. Install from apt(very old)
      • $ sudo apt install mongodb
  7. redis: for project development.
    • $ wget http://download.redis.io/releases/redis-3.2.5.tar.gz
    • $ tar -xf redis-3.2.5.tar.gz
    • $ mv redis-3.2.5 redis
    • $ cd redis
    • $ make
    • $ sudo /opt/redis/src/redis-server /usr/bin/redis-server
    • $ sudo /opt/redis/src/redis-cli /usr/bin/redis-cli
  8. nginx
    • $ apt install nginx
  9. mkdir: Built up folders
    • $ cd ~
    • $ mkdir assets backups cooperations data get-started projects workspace
  10. forever: run programs(scripts) forever without unexpected exit.
    • $ npm install forever

Set up a HTTP static server

Run one of the following command to start a server that serve static files.

  • $ python -m SimpleHTTPServer 3000
  • $ ruby -run -ehttpd ./ -p3000
